Documentation

This section provides documentation for the Gemini AI Chatbot.

Getting Started

To use the chatbot, simply type your message into the input field and press 'Send' or hit 'Enter'. The AI will respond in real-time.

Features

  • Gemini API Integration for intelligent responses.
  • Modern, green-themed UI design.
  • Responsive layout for various devices.
  • Real-time conversational experience.

API Key

The chatbot utilizes the Google Gemini API. Ensure your API key is correctly configured in script.js.
